# Le Foin Framework

> Research Meets Entertainment

Le Foin Framework is a modern publishing framework built on WordPress,
designed for researchers, writers, indie creators and digital studios.

Instead of being a traditional blog theme,
Le Foin aims to become a complete content ecosystem.

---

## Features

- Modern Homepage
- Archive Template
- Single Article Template
- Related Articles
- Search Page
- Custom 404 Page
- Responsive Design
- GitHub-first Development

---
